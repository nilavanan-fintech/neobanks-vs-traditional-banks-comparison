# Primary NPS Survey — Draft Questions

Target n=50–100 respondents, split roughly evenly between Jupiter and HDFC Bank users. Deploy via Google Forms per `README.md` §3.

1. Which app do you primarily use? (Jupiter / HDFC Bank / Both / Other — specify)
2. On a scale of 0–10, how likely are you to recommend this app to a friend or colleague? *(the standard NPS question — used to compute promoter/detractor split)*
3. How long have you been using this app? (<3 months / 3–12 months / 1–3 years / 3+ years)
4. What do you use it for most? (Savings / UPI payments / Cards / FD / Loans / Investments / Insurance / Forex)
5. What's the single best thing about this app? (open text)
6. What's the single most frustrating thing about this app? (open text)
7. Have you contacted customer support in the last 3 months? If yes, how would you rate the experience? (1–5)
8. Compared to your previous/other bank's app, is this better, worse, or about the same? (open text, optional)
9. Age bracket (18–24 / 25–34 / 35–44 / 45–54 / 55+)
10. City tier (Tier 1 / Tier 2 / Tier 3 / Rural)

## Scoring note
- Promoters = respondents scoring 9–10 on Q2. Detractors = 0–6. Passives = 7–8.
- Proxy NPS = %Promoters − %Detractors, computed separately per app, to compare against the app-store-derived proxy NPS in `data/raw/app_reviews_summary.md`.
